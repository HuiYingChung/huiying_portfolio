# Huiying Chung — Portfolio

Personal portfolio website for Huiying Chung, a UX/UI and front-end designer.

Live site: [huiyingchung.github.io](https://huiyingchung.github.io/)

---

## About

A personal portfolio showcasing UX case studies and front-end projects. Built with semantic HTML, vanilla CSS, and JavaScript — no frameworks or build tools.

This site was built in collaboration with AI. The design decisions, content, and direction are mine.

---

## Case Studies

| Project | Focus |
|---|---|
| **ICC Houston Website Redesign** | Information Architecture, Stakeholder Interviews — live at icc-houston.org |
| **KitchenAid Espresso Machine Redesign** | UX Analysis, Interface Design, Prototyping |
| **Tensegrity Instruction Set** | UX Writing, Usability Testing, Instructional Design |
| **Little Cafe** | Front-End (HTML/CSS/JS), UX, E-commerce concept |
| **South Trail Journal** | Front-End, Multi-page site, GitHub Pages deployment |

---

## Tech Stack

- HTML5 (semantic)
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript
- Google Fonts — Syne + Figtree
- Deployed on GitHub Pages

---

## Features

- Dark / light mode toggle
- Responsive layout with mobile hamburger menu
- Scroll-triggered fade-in animations
- Back-to-top button
- Contact form page
- W3C-validated markup

---

## Local Development

No build step needed. Open `index.html` directly in a browser, or use a local server:

```bash
npx serve .
```
